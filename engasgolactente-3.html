Obstrução total

Se o bebê não consegue chorar ou tossir, apresenta dificuldade para respirar ou possui os lábios de coloração azulada, trata-se de um OBSTRUÇÃO TOTAL. Para ajudá-lo, você deve seguir as seguintes etapas.

Prepare-se

Prepare-se para realizar as manobras de desobstrução de vias aéreas no lactente. Para isso, sente-se em uma superfície rígida, preferencialmente em uma cadeira, para obter melhor apoio. Além disso, se for possível, retire as roupas que cobrem o tórax do bebê.
